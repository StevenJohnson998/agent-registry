# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Agent Registry, please report it responsibly.

**Do NOT open a public issue for security vulnerabilities.**

Instead, please email: [your-email] with:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and provide updates as we work on a fix.
